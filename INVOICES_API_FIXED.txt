INVOICE ITEMS MANAGEMENT:
✅ GET    /api/invoices/:id/items
   └─ Fetch all items for an invoice
   └─ Returns: Array of invoice items

✅ POST   /api/invoices/:id/items
   └─ Create a new line item for an invoice
   └─ Body: { item_name, description, quantity, rate, discount_percentage, tax_percentage }
   └─ Returns: Created item object

✅ PUT    /api/invoices/:invoiceId/items/:itemId
   └─ Update an invoice line item
   └─ Body: { item_name, description, quantity, rate, discount_percentage, tax_percentage }
   └─ Returns: Updated item object

✅ DELETE /api/invoices/:invoiceId/items/:itemId
   └─ Delete a line item from invoice
   └─ Returns: Success message

INVOICE METRICS & REPORTING:
✅ GET    /api/invoices/metrics/summary
   └─ Get invoice statistics (total, count, average)
   └─ Returns: { summary: {...}, byStatus: [...] }

✅ GET    /api/invoices/status/breakdown
   └─ Get invoice count and total by status
   └─ Returns: Array with status breakdown

INVOICE RELATIONSHIPS:
✅ GET    /api/companies/:companyId/invoices
   └─ Get all invoices for a specific company
   └─ Query params: skip, limit, status
   └─ Returns: Array of invoices

✅ GET    /api/deals/:dealId/invoices
   └─ Get all invoices linked to a specific deal
   └─ Query params: skip, limit, status
   └─ Returns: Array of invoices

✅ POST   /api/invoices/:invoiceId/link-to-deal/:dealId
   └─ Link an invoice to a deal/project
   └─ Returns: Updated invoice

✅ POST   /api/invoices/:invoiceId/link-to-client/:clientId
   └─ Link an invoice to a client/company
   └─ Returns: Updated invoice

INVOICE OPERATIONS:
✅ POST   /api/invoices/:id/refund
   └─ Process a refund for an invoice
   └─ Body: { refund_amount, refund_date, reason }
   └─ Returns: Updated invoice with refund status

✅ DELETE /api/invoices/:id
   └─ Delete an invoice and its items
   └─ Returns: Success message
