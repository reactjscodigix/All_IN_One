                    🎯 HOW TO USE THE SYSTEM

1. CREATE CAMPAIGN
   → Go to Campaign page
   → Click "+ Add Campaign"
   → Set budget, dates, type
   → System generates lead source field

2. GENERATE LEADS
   → Leads auto-generated from campaign activities
   → Each lead shows: name, email, lead_source, status
   → Mark as "Qualified" when ready

3. CONVERT TO DEAL
   → Click "Convert to Deal" on qualified lead
   → System creates deal in pipeline
   → Assign to sales rep
   → Set probability based on stage

4. TRACK IN PIPELINE
   → View pipeline page
   → See deal progress by stage
   → Move deals through stages
   → Track probability to close

5. ANALYZE CAMPAIGN ROI
   → Campaigns → Generated Leads → Qualified → Deals Won
   → ROI = Deals Won Value / Campaign Budget
   
   Example:
   ├─ Google Ads ($5,000) → 2 qualified → 1 won ($120k)
   ├─ ROI = $120,000 / $5,000 = 24x return! 🎉
