DEFENSIVE RESPONSE HANDLING:
================================================================================

All pages now check for multiple API response formats:

```javascript
let itemsList = data;
if (!Array.isArray(data)) {
  itemsList = data?.data || data?.[entityName] || [];
}

if (Array.isArray(itemsList)) {
  // Process and display
} else {
  setItems([]);
}
```

This handles:
- Plain arrays: [{ id: 1, ... }]
- Wrapped objects: { data: [{ id: 1, ... }] }
- Named containers: { items: [{ id: 1, ... }] }
